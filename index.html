Drinker

  • Android

  • Unity

  • C#

Drinker is primarily a timer for cocktail parties and feasts. Minimalistic app with a clean design that lets you quickly start a timer. It features a large clock interface on the home page which you can interact with to set the duration. The application contains a variety of themes and funny ringtones that you can change. Recommended for parties and feasts with friends.

For this application i used the game engine unity and the basic knowledge of the C # programming language. All melodies and effects were written in Logic Pro X. The design of the application was developed in a graphic editor.
